# 双方保密协议 · 标准条款

1. <span class="header_2" id="1">协议构成</span>  本《双方保密协议》由本标准条款及双方签署的封面页（以下合称"本协议"）构成。为<span class="coverpage_link">保密用途</span>，任何一方（"披露方"）均可能向另一方（"接收方"）披露或提供：（一）披露方标明"保密""专有"或类似字样的信息；或（二）根据信息性质及披露情形应被合理认为具有保密性质的信息（以下合称"保密信息"）。保密信息包括但不限于技术资料、经营信息、产品设计与规划、需求文档、价格、安全与合规文件、技术成果与专有技术，亦包括双方磋商的存在及进展情况和封面页所载信息。

2. <span class="header_2" id="2">保密信息的使用与保护</span>  接收方应当：（一）仅为<span class="coverpage_link">保密用途</span>使用保密信息；（二）未经披露方事先书面同意，不得向任何第三方披露保密信息，但接收方可以向其因该用途确有必要知悉的员工、代理人、顾问、承包商及其他代表披露，前提是该等人员受到不低于本协议标准的保密义务约束，且接收方对其遵守本协议的情况承担责任；（三）以不低于其保护自身同类信息的措施保护保密信息，且不得低于合理注意义务。

3. <span class="header_2" id="3">除外情形</span>  接收方能够举证证明下列情形之一的信息，不适用本协议项下的保密义务：（一）非因接收方过错已经公开的；（二）接收方在披露前已经合法知悉且不负有保密义务的；（三）接收方从对该信息不负有保密义务的第三方合法获得的；（四）接收方未使用、未参考保密信息而独立开发取得的。

4. <span class="header_2" id="4">依法披露</span>  接收方根据法律、行政法规、监管机关要求或司法程序必须披露保密信息的，可以在必需的范围内予以披露，但应当在法律允许的范围内提前书面通知披露方，并在披露方承担费用的前提下合理配合披露方寻求保密处理。

6. <span class="header_2" id="6">返还与销毁</span>  本协议期满、终止或经披露方书面要求时，接收方应当停止使用并销毁或返还全部保密信息，并应要求以书面形式确认；接收方可按其正常备份或档案留存制度或法律要求留存，本协议对留存信息继续适用。

7. <span class="header_2" id="7">权利保留</span>  披露方保留其对保密信息享有的全部知识产权及其他权利；披露不构成任何许可、转让或授予。

8. <span class="header_2" id="8">不作保证</span>  所有保密信息均按"现状"提供，披露方不作任何明示或默示保证。

9. <span class="header_2" id="9">违约责任</span>  接收方违约给披露方造成损失的，应赔偿披露方因此遭受的损失（含合理的调查取证费用、公证费、鉴定费及律师费）；侵犯商业秘密的，披露方有权依照《中华人民共和国反不正当竞争法》等追究法律责任；金钱赔偿不足以弥补损害的，披露方有权依法申请行为保全、禁令等救济。
